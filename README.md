# Network-Mapping-with-Nmap
Network Mapping with Nmap is a cybersecurity mini project conducted in a virtual lab using Kali Linux and VirtualBox. It demonstrates host discovery, port scanning, service and version detection, OS fingerprinting, and network reconnaissance using Nmap, showcasing practical skills in safe and controlled network analysis.
## Objective

Perform network discovery and identify active hosts, open ports, services, and operating systems inside a virtual lab.

## Tools

- Kali Linux
- Nmap
- Ubuntu
- VirtualBox

## Commands Used

nmap -sn
nmap
nmap -sV
nmap -O
nmap -A

## Result

Successfully identified:

Active hosts

Open ports

Running services

Operating system

## Learning Outcome

Understanding network reconnaissance using Nmap in a safe virtual environment.
